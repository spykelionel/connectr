## Quick Start

1. **Install Dependencies**

   ```bash
   npm install
   # or run the batch file on Windows
   install-deps.bat
   ```

2. **Start Development Server**

   ```bash
   npm run dev
   ```

3. **Build for Production**

   ```bash
   npm run build
   ```
